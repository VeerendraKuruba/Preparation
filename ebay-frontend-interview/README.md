# eBay Frontend Engineer Interview Prep

Organized by round type. Work through each file in order.

## Interview Process (4–5 rounds)
1. Recruiter screen (30 min)
2. Online Assessment — CodeSignal/HackerRank (45–90 min)
3. Technical Phone Screen (45 min × 2)
4. Onsite Loop (1 day: Coding + System Design + Frontend Tech + Behavioral)

**Difficulty:** LeetCode Easy–Medium. Nothing Hard or brain-teaser level (confirmed by multiple candidates).
